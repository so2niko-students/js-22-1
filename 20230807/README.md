# Інструкція

### Інсталяція та стартове налаштування
1. Інсталював шаблон аплікації через термінал
`npx create-react-app ./`

2. Видалив "зайві" файли:

- readme.md
- src/app.test.js
- src/setupTests.js
- src/reportWebVitals.js
- src/logo.svg
- public/robots.txt
- public/manifest.json
- public/logo512.png
- public/logo192.png
- public/favicon.ico

3. Відредагував public/index.html

4. Відредагував src/index.js

5. Відредагував src/app.js

### Розробка

6. Запуск аплікації через термінал
`npm run start`
